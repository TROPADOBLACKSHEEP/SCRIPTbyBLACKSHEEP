<h1>APLICATIVO IPTV GERENCIADOR DE SCRIPTS</h1>
<p>Este aplicativo foi desenvolvido para facilitar o gerenciamento e download de scripts, combos e addons relacionados ao IPTV, proporcionando uma interface intuitiva e acessível.</p>

<h2>FUNCIONALIDADES</h2>
<ul>
    <li><strong>VERIFICAÇÃO DE ATUALIZAÇÕES:</strong> O aplicativo verifica automaticamente se há novas versões do script disponível.</li>
    <li><strong>DOWNLOAD DE RECURSOS:</strong> Permite baixar scripts, combos e addons, armazenando-os em uma pasta dedicada.</li>
    <li><strong>INTERFACE AMIGÁVEL:</strong> Navegue facilmente entre as funcionalidades e recursos disponíveis.</li>
    <li><strong>INTEGRAÇÃO COM TELEGRAM:</strong> Acesse rapidamente a comunidade para suporte e novidades.</li>
</ul>

<h2>PRÉ-REQUISITO</h2>
<p>Antes de executar o aplicativo, certifique-se de ter o <strong>Python 3.11</strong> ou superior instalado em seu sistema, junto com os seguintes pacotes:</p>
<ul>
    <li><strong>requests</strong></li>
    <li><strong>kivy</strong></li>
</ul>

<h2>INSTALAÇÃO DOS PACOTES NECESSÁRIOS</h2>
<p>O aplicativo verifica automaticamente se os pacotes necessários estão instalados. Caso contrário, ele tenta instalá-los utilizando o pip.</p>

<h2>Estrutura do Código</h2>
<p>O aplicativo é estruturado em várias classes principais:</p>
<ul>
    <li><strong>MeuApp:</strong> Classe principal que inicia o aplicativo e gerencia as telas.</li>
    <li><strong>GerenciadorTela:</strong> Controla a navegação entre diferentes telas.</li>
    <li><strong>TelaChecker:</strong> Tela principal que verifica atualizações e permite o download de scripts.</li>
    <li><strong>TELATERMINAL:</strong> Tela que exibe um terminal para feedback de operações.</li>
</ul>

<h2>DOWNLOAD e ATUALIZAÇÕES</h2>
<p>O aplicativo baixa a versão mais recente do script diretamente do GitHub. Caso uma nova versão seja detectada, o usuário é notificado e pode optar por atualizar.</p>
